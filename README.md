# File Encryption/Decryption

A desktop GUI app with react and node for encrypting a file using AES-256-GCM and PBKDF2.

## How to run

Download Node.js, go into the project root directory and run:

```bash
  npm install
```
Run this in the terminal to open the app and run Electron runtime

```bash
  npm run build
  npm run electron
```
Run this to build the app and get the .exe

```bash
  npm run build
  npm run dist
```
